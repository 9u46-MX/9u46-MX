[Solution] CF1372D

发布于 2021-02-19 | 分类于 solution | 4分钟 | 739字数

CF1372D.
Luogu blog.

有一个环 a1⋯na_{1 \cdots n}a1⋯n​（2∤n2 \nmid n2∤n），每次操作可以选择环上一点，使这一点的值变为相邻两点值的和，并删去相邻两点。

经过一些操作（n−12\frac {n - 1} 22n−1​ 次）后，环会变为一点。求这一点可能的最大值。

1⩽n<2×1051 \leqslant n < 2 \times 10^51⩽n<2×105，0⩽ai⩽1090 \leqslant a_i \leqslant 10^90⩽ai​⩽109。

本题解省略取模操作。

首先，我们发现，最终答案必然是 aaa 中一些位置的和。

我们考虑，一次操作，相当于删去一个位置的值。这个位置可能是一个 aia_iai​，也可能因为之前的操作而是一些 aia_iai​ 的和。

我们令 bib_ibi​ 为 aia_iai​ 是否被删除，aia_iai​ 始终不变，选中一个 bi=0b_i = 0bi​=0 的位置相当于选中了和这个位置“绑定”在一起的一些位置。

第一次操作，选中 ppp，bp←1b_p \leftarrow 1bp​←1。

后面的操作，选中 qqq，若 bq−1=1b_{q - 1} = 1bq−1​=1，则 q−2q - 2q−2 也被选中，若 bq+1=1b_{q + 1} = 1bq+1​=1，则 q+2q + 2q+2 也被选中，这是一个递归的过程。

通过模拟，我们发现，最后 01⋯1001 \cdots 1001⋯10 的块必然满足 111 的数量为奇数。易证。

对于每块 01⋯1001 \cdots 1001⋯10，我们考虑进行相同数量的操作（选中位置不同），也可以使这一块变为 0101⋯0100101 \cdots 0100101⋯010。

而 ai⩾0a_i \geqslant 0ai​⩾0，故我们最终的 bbb 应该有 n−12\frac{n - 1}22n−1​ 个位置是 111，其余位置是 000，且为 111 的位置互不相邻。

于是，问题转化为，选中互不相邻的 n−12\frac{n - 1}22n−1​ 个 aia_iai​，求最小的和。

比较简单的问题。

本人的做法是破环为链，对奇数位与偶数位分别做前缀和。当然还有别的方法。

时间复杂度 O(n)\mathcal O(n)O(n)。

（和代码思路略有不同，代码是直接删去互不相邻的 n−12\frac{n - 1}22n−1​ 个 aia_iai​，求最大的剩余和。）

# include <cstdio>

namespace Main {
    namespace Source {
        typedef unsigned int uint;
        typedef long long unsigned int llu;
        static inline const llu max(const llu a, const llu b) { return a < b ? b : a; }
        static inline const void amax(llu& a, const llu b) { a = max(a, b); }
    }
    using namespace Source;
    enum { N = (const uint)2e5 };
    static uint n;
    static uint a[N * 2];
    static llu ans;
    static inline const void main() {
        scanf("%u", &n);
        for (register uint i(1); i <= n; ++i) scanf("%u", &a[i]);
        for (register uint i(1); i <= n; ++i) a[n + i] = a[i];
        static llu s;
        s = 0; for (register uint i(1); i <= n; i += 2) s += a[i];
        amax(ans, s);
        for (register uint i(n + 2); i < n * 2; i += 2) amax(ans, s = s + a[i] - a[i - n - 1]);
        s = 0; for (register uint i(2); i <= n + 1; i += 2) s += a[i];
        amax(ans, s);
        for (register uint i(n + 3); i <= n * 2; i += 2) amax(ans, s = s + a[i] - a[i - n - 1]);
        printf("%llu\n", ans);
    }
}

signed int main() { Main::main(); return 0; }
