A - escape

KHIN's expected

考虑应该把哪个字符移到第一位。
显然应该是可以移动的字符中最小的，如果有多个选取最靠前的一个（为后面留交换次数）。
贪心即可，O(n2)O(n^2)O(n2)。由于有 95pts，故暂时不必优化。
95pts / 95pts。

real Solution

平衡树即可。splay tree 比较方便。
担心调不出来而没写= =

B - ride

考虑换根 DP。
对于每个点，记录以她作为最高点有多少下降序列。简单换根即可。
然后统计即可。
O(n)O(n)O(n)。
100pts / 100pts。
