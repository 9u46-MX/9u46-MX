[Solution] CF838C

发布于 2021-02-04 | 分类于 solution | 11分钟 | 2058字数

CF838C.
Luogu blog.

两人轮流操作一个长度为 nnn 的字符集大小为 kkk 的字符串。每次将字符串重新排列或删去一个字符，不能和之前的字符串相同。不能操作者失败。

问有多少种字符串可以使得先手必胜，对质数 ppp 取模。

空串是被允许的，但显然空串无法再操作。

1⩽n⩽2.5×1051 \leqslant n \leqslant 2.5 \times 10^51⩽n⩽2.5×105，1⩽k⩽261 \leqslant k \leqslant 261⩽k⩽26，108⩽p⩽109+10010^8 \leqslant p \leqslant 10^9 + 100108⩽p⩽109+100。

首先，我们考虑，什么样的字符串先手必胜。设 sss 是一个可能出现在游戏中的字符串。

我们读题后发现，sss 是否先手必胜仅与 sss 中每个字符的出现次数 c1⋯kc_{1 \cdots k}c1⋯k​ 有关，与 sss 中字符的排列顺序无关。

而字符串 sss 的不同排列数 h(s)h(s)h(s) 为

h(s)=∣s∣!∏i=1kci!.h(s) = \frac{|s|!}{\prod_{i = 1}^k c_i!}. h(s)=∏i=1k​ci​!∣s∣!​.

所以，如果字符串 sss 的其他排列都没有出现过（即游戏开始时或删去字符后），删除操作被后手进行，当且仅当 2∣h(s)2 \mid h(s)2∣h(s) 且 删除任何一个字符 都会到达一个 后手必败的局面。

所以我们可以列出下面四种情况：

  • 2∤h(s)2 \nmid h(s)2∤h(s)，删除字符后 可能 到达后手必胜局面：
    先手愉快地删除字符，先手必胜。
  • 2∤h(s)2 \nmid h(s)2∤h(s)，删除字符后 不可能 到达后手必胜局面：
    先手无奈地删除字符，先手必败。
  • 2∣h(s)2 \mid h(s)2∣h(s)，删除字符后 可能 到达后手必胜局面：
    先手愉快地删除字符，先手必胜。
  • 2∣h(s)2 \mid h(s)2∣h(s)，删除字符后 不可能 到达后手必胜局面：
    后手无奈地删除字符，先手必胜。

所以，当 2∣h(s)2 \mid h(s)2∣h(s) 时，先手必胜；当 2∤h(s)2 \nmid h(s)2∤h(s) 时，先手必胜当且仅当删除字符后 可能 到达后手必胜局面。

我们考虑 2∤h(s)2 \nmid h(s)2∤h(s) 时，是否能达到后手必胜局面。

考虑选择一个质因数分解后 222 的次数最小的 cic_ici​，则 h(s′)=h(s)⋅ci∣S∣h(s') = h(s) \cdot \frac{c_i}{|S|}h(s′)=h(s)⋅∣S∣ci​​，显然 cic_ici​ 的质因数分解后 222 的次数不大于 ∣S∣|S|∣S∣ 分解后 222 的次数，故操作后 2∤h(s′)2 \nmid h(s')2∤h(s′)，另一方仍需要如此操作。

所以，当 2∤h(s)2 \nmid h(s)2∤h(s) 时，先手必胜当且仅当 2∤∣s∣2 \nmid |s|2∤∣s∣。

综上，先手必胜等价于 2∤n∨(2∣n∧2∣h(s))2 \nmid n \lor (2 \mid n \land 2 \mid h(s))2∤n∨(2∣n∧2∣h(s))。故当 2∤n2 \nmid n2∤n 时输出 knk^nkn 即可。

当 2∣n2 \mid n2∣n 时，我们应输出 ∣s∣=n∧2∣h(s)|s| = n \land 2 \mid h(s)∣s∣=n∧2∣h(s) 的字符串数量。

我们设 b(x)b(x)b(x) 为 x!x!x! 质因数分解后 222 的次数，即

b(x)=∑i∈N+⌊x2i⌋.b(x) = \sum_{i \in \mathbf N_+} \lfloor \frac x {2^i} \rfloor. b(x)=i∈N+​∑​⌊2ix​⌋.

则答案应为满足 ∑i=1kb(ci)<b(n)\sum_{i = 1}^k b(c_i) < b(n)∑i=1k​b(ci​)<b(n) 的字符串数。该关系等价于

∀j∈N+,∑⌊ci2j⌋<⌊n2j⌋.\forall j \in \mathbf N_+, \sum \lfloor \frac {c_i}{2^j} \rfloor < \lfloor \frac n {2^j} \rfloor. ∀j∈N+​,∑⌊2jci​​⌋<⌊2jn​⌋.

而显然

∀j∈N+,∑⌊ci2j⌋⩽⌊n2j⌋,\forall j \in \mathbf N_+, \sum \lfloor \frac {c_i}{2^j} \rfloor \leqslant \lfloor \frac n {2^j} \rfloor, ∀j∈N+​,∑⌊2jci​​⌋⩽⌊2jn​⌋,

所以我们只需要求出满足

∀j∈N+,∑⌊ci2j⌋=⌊n2j⌋\forall j \in \mathbf N_+, \sum \lfloor \frac {c_i}{2^j} \rfloor = \lfloor \frac n {2^j} \rfloor ∀j∈N+​,∑⌊2jci​​⌋=⌊2jn​⌋

的字符串数 ttt，答案即为 kn−tk^n - tkn−t。

我们令 jjj 为 nnn 二进制最高位，则 ∑⌊ci2j⌋=⌊n2j⌋\sum \lfloor \frac {c_i}{2^j} \rfloor = \lfloor \frac n {2^j} \rfloor∑⌊2jci​​⌋=⌊2jn​⌋ 等价于所有 cic_ici​ 中有且仅有一个在 nnn 最高位位 111。

我们再令 jjj 为 nnn 的第二位，结合最高位的结论，∑⌊ci2j⌋=⌊n2j⌋\sum \lfloor \frac {c_i}{2^j} \rfloor = \lfloor \frac n {2^j} \rfloor∑⌊2jci​​⌋=⌊2jn​⌋ 等价于：若 nnn 第二位为 000，所有 cic_ici​ 在该位均为 000；否则有且仅有一个 cic_ici​ 在该位为 111。

以此类推，所有 cic_ici​ 应满足

∀1⩽i<j⩽k,cibitand⁡cj=0.\forall 1 \leqslant i < j \leqslant k, c_i \operatorname{bitand} c_j = 0. ∀1⩽i<j⩽k,ci​bitandcj​=0.

所以，我们将问题转化为，nnn 的二进制一共有 lll 位为 111，我们把这 lll 位“分给” kkk 个字符，求所有方案中下式之和即为 ttt：

n!∏i=1kci!.\frac{n!}{\prod_{i = 1}^k c_i!}. ∏i=1k​ci​!n!​.

于是，我们有一个简单的 DP：设 fi,jf_{i, j}fi,j​ 为 nnn 用去了 iii，jjj 个字符已分配完成时的

fi,j=∑kbitor⁡i=ifk,j−1⋅1(i−k)!.f_{i, j} = \sum_{k \operatorname{bitor} i = i} f_{k, j - 1} \cdot \frac{1}{(i - k)!}. fi,j​=kbitori=i∑​fk,j−1​⋅(i−k)!1​.

我们考虑，本质相同的方案被计算了多次。我们考虑 fi,jf_{i, j}fi,j​ 强制选择 lowbit⁡(i)\operatorname{lowbit}(i)lowbit(i)，若最终使用了 xxx 个字符，则答案再乘 A⁡(x,k)\operatorname A(x, k)A(x,k)。

实现上，因为 f0,if_{0, i}f0,i​ 表示 iii 个字符的出现次数为 000，如果选了 f0,if_{0, i}f0,i​ 则必然仅有 k−ik - ik−i 个字符瓜分 nnn，故可以将 f0,if_{0, i}f0,i​ 设为 A⁡(n−i,k)\operatorname A(n - i, k)A(n−i,k)。

# include <cstdio>
# include <cstring>

namespace Main {
    namespace Source {
        typedef short unsigned int hu;
        typedef unsigned int uint;
        static inline const uint qpow(uint b, uint e, const uint mod) {
            uint r(1);
            while (e) {
                if (e bitand 1) r = 1ull * r * b % mod;
                b = 1ull * b * b % mod, e >>= 1;
            }
            return r;
        }
        static inline const uint finv(const uint x, const uint p) { return qpow(x, p - 2, p); }
        static uint *invf, *fact;
        struct __FILE {
            __FILE() {}
            compl __FILE() { if (invf) delete[]invf; if (fact) delete[]fact; }
        } ____FILE;
        static inline const void init(const uint n, const uint p) {
            if (not n) return;
            if (invf) delete[]invf; if (fact)delete[]fact;
            invf = new uint [n], fact = new uint [n], invf[0] = 1, fact[0] = 1;
            for (register uint i(1); i < n; ++i) fact[i] = 1ull * fact[i - 1] * i % p;
            invf[n - 1] = finv(fact[n - 1], p);
            for (register uint i(n - 2); i < n; --i) invf[i] = 1ull * invf[i + 1] * (i + 1) % p;
        }
        static inline const uint lowbit(const uint x) { return x bitand ~x + 1; }
    }
    using namespace Source;
    enum { N = 250000, K = 26 };
    static uint n, p; static hu k;
    static uint _f[N + 1][K + 1]; // f(i, j): i position, j letter, sum prod inv(fact)
    // f(0, i) = A(k, k - i)
    // f(i, j) = sum{ 0 <= k <= i & k in i } f(k, j) / (i - k)!
    // ans = p^n - f(n, k)
    static const uint f(const uint a, const hu b) {
        static bool g(false);
        if (not g) {
            for (register uint i(0); i <= n; ++i)
                for (register uint j(0); j <= k; ++j) _f[i][j] = p;
            for (hu i(0); i <= k; ++i) {
                _f[0][i] = fact[n];
                for (hu j(0); j < k - i; ++j) _f[0][i] = 1ull * _f[0][i] * (k - j) % p;
                // printf("%hu %u\n", i, _f[0][i]);
            }
            g = true;
            # ifdef DJ
                puts("count");
            # endif
        }
        if (_f[a][b] < p) return _f[a][b];
        if (a and !b) return 0;
        const uint t(a xor lowbit(a)); _f[a][b] = 1ull * f(0, b - 1) * invf[a] % p;
        for (register uint i(t); i; i = i - 1 & t)
            _f[a][b] = (_f[a][b] + 1ull * f(i, b - 1) * invf[a ^ i] % p) % p;
        return _f[a][b];
    }
    static inline const void main() {
        scanf("%u%hu%u", &n, &k, &p);
        if (n % 2) return (const void)printf("%u\n", qpow(k, n, p));
        init(n + 1, p);
        # ifdef DJ
            for (register uint i(0); i <= n; ++i) {
                for (hu j(0); j <= k; ++j) printf("%u ", f(i, j));
                putchar('\n');
            }
        # endif
        printf("%u\n", (qpow(k, n, p) + p - f(n, k)) % p);
