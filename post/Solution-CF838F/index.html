[Solution] CF838F

发布于 2021-02-13 | 分类于 solution | 5分钟 | 970字数

CF838F.
Luogu blog.

一个袋子里有 nnn 个红色或黑色的球，每抽一个球花费为 ccc。

袋子中有 iii 个红球的概率为 10−6pi10^{-6}p_i10−6pi​（记为 qiq_iqi​）。每抽到一个红球的收益为 111。你可以根据实际情况随时停止抽球。

求最大期望收益。

n⩽104n \leqslant 10^4n⩽104，0⩽c,qi⩽10 \leqslant c, q_i \leqslant 10⩽c,qi​⩽1。

考虑 DP。

首先，要求什么考虑设什么。fi,jf_{i, j}fi,j​：已经抽了 iii 个球，抽出 jjj 个红球，此后 的最大期望收益。

设此时下一个球是红球的概率为 gi,jg_{i, j}gi,j​。显然 fff 的转移方程如下：

fi,j=max⁡(gi,j(fi+1,j+1+1)+(1−gi,j)fi+1,j−c,0).f_{i, j} = \max(g_{i, j}(f_{i + 1, j + 1} + 1) + (1 - g_{i, j})f_{i + 1, j} - c, 0). fi,j​=max(gi,j​(fi+1,j+1​+1)+(1−gi,j​)fi+1,j​−c,0).

考虑如何求出 gi,jg_{i, j}gi,j​。ggg 不太容易 DP，考虑再设一个辅助值。

hi,jh_{i, j}hi,j​：在抽了 iii 个球的条件下，抽出 jjj 个红球的概率。

显然，若抽了 iii 个球，抽出 jjj 个红球，那么若再抽一次，只可能抽出 jjj 或 j+1j + 1j+1 个红球。如下两点显然：

  • 若抽了 i+1i + 1i+1 个球，抽出 jjj 个红球，则最后一次抽出 黑球 的概率为 i−j+1i+1\frac{i - j + 1}{i + 1}i+1i−j+1​；
  • 若抽了 i+1i + 1i+1 个球，抽出 j+1j + 1j+1 个红球，则最后一次抽出 红球 的概率为 j+1i+1\frac{j + 1}{i + 1}i+1j+1​。

又根据 hi+1,jh_{i + 1, j}hi+1,j​、hi+1,j+1h_{i + 1, j + 1}hi+1,j+1​，可得到 hhh 的转移方程如下：

hi,j=hi+1,j+1⋅j+1i+1+hi+1,j⋅i−j+1i+1.h_{i, j} = h_{i + 1, j + 1} \cdot \frac{j + 1}{i + 1} + h_{i + 1, j} \cdot \frac{i - j + 1}{i + 1}. hi,j​=hi+1,j+1​⋅i+1j+1​+hi+1,j​⋅i+1i−j+1​.

现在问题只剩下了 ggg 的转移方程。

我们考虑，在抽了 iii 个球的条件下抽出 jjj 个红球的概率为 hi,jh_{i, j}hi,j​；在抽了 i+1i + 1i+1 个球的条件下，前 iii 个球中抽出 jjj 个红球且最后一个球是红球的概率为 hi+1,j+1⋅j+1i+1h_{i + 1, j + 1} \cdot \frac{j + 1}{i + 1}hi+1,j+1​⋅i+1j+1​。故我们可以得到 ggg 的表达式如下：

gi,j=hi+1,j+1(j+1)hi,j(i+1).g_{i, j} = \frac{h_{i + 1, j + 1}(j + 1)}{h_{i, j}(i + 1)}. gi,j​=hi,j​(i+1)hi+1,j+1​(j+1)​.

我们发现 f,g,hf, g, hf,g,h 分别可以滚动数组或直接省略掉第一维。

时间复杂度 O(n2)\mathcal O(n^2)O(n2)，空间复杂度 O(n)\mathcal O(n)O(n)。

尤其注意 000 作分母的情况。

看题解没看懂结果自己做出来了 /jk

# include <cstdio>

namespace Main {
    namespace Source {
        typedef short unsigned int hu;
        typedef unsigned int uint;
        typedef double lf;
        static inline const lf max(const lf a, const lf b) { return a < b ? b : a; }
    }
    using namespace Source;
    enum { N = 10000 };
    static hu n;
    static uint p[N + 1]; static lf q[N + 1];
    static lf c; static uint x;
    static lf f[N + 1], g[N + 1], h[N + 1];
    static inline const void main() {
        scanf("%hu%u", &n, &x), c = x / 1e6;
        for (hu i(0); i <= n; ++i) scanf("%u", &p[i]), q[i] = p[i] / 1e6;
        for (hu i(0); i < n; ++i) {
            if (not(h[i] = q[i] * (n - i) / n + q[i + 1] * (i + 1) / n)) continue;
            g[i] = q[i + 1] * (i + 1) / (q[i] * (n - i) + q[i + 1] * (i + 1)),
            f[i] = max(g[i] - c, 0);
        }
        for (hu i(n - 2); i < n; --i) for (hu j(0); j <= i; ++j) {
            lf pr(h[j + 1] * (j + 1) / (i + 1)), pb(h[j] * (i + 1 - j) / (i + 1));
            if (not(h[j] = pr + pb)) continue;
            g[j] = pr / h[j];
            f[j] = max(g[j] * (f[j + 1] + 1) + (1 - g[j]) * f[j] - c, 0);
        }
        printf("%.12lf\n", f[0]);
    }
}

signed int main() { Main::main(); return 0; }
